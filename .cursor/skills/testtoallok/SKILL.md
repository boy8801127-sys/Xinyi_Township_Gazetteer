# 測試通過才完成 (Test to All OK)

## 使用時機

當使用者呼叫此技能（例如輸入 `/testtoallok`）或明確要求「需測試可順暢執行才能完成」時套用。

## 必須執行的步驟

1. **執行相關測試**
   - 若有測試指令（如 `npm test`、`pytest`、`cargo test` 等），在專案根目錄執行。
   - 若無既有測試，至少執行能啟動或建置專案的指令（如 `npm run build`、`npm start`、`python main.py` 等），確認可正常運行。

2. **確認結果**
   - 測試或執行必須**成功、無錯誤**，才可將任務視為完成。
   - 若有失敗：修正問題後重新執行，直到通過為止。

3. **完成條件**
   - 僅在「測試／執行已通過、可順暢運行」後，才回報任務完成。
   - 若無法通過，需說明失敗原因與建議修正方向，不得宣稱已完成。
