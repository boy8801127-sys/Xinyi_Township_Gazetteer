# 信義鄉志格式檢核

依據專案內 `This_plan/信義鄉志服務建議書.pdf` 之「編纂凡例」與「撰寫格式」進行檢視。

## 使用時機

- 使用者呼叫本技能（如 `/gazetteer-format-check`）。
- 使用者要求「依鄉志格式檢視」、「檢核志書格式」、「檢查是否符合信義鄉志規範」時。

## 檢核前準備

1. 確認要檢核的範圍：指定檔案、資料夾或貼上的內容。
2. 必要時先閱讀 `This_plan/信義鄉志服務建議書.pdf` 中「（六）編纂凡例」「（七）撰寫格式」以取得最新規範。

---

## 一、編纂凡例檢核

- [ ] **語文**：全文正體中文，「臺灣」之「臺」為繁體「臺」；引用原文為簡體者照引。
- [ ] **紀年**：以各政權紀元為主、西元加註，如民國 34 年（1945）、光緒元年（1875）、大正元年（1912）；時期用「史前、荷治、明鄭、清治、日治、戰後」等學界慣用詞。
- [ ] **引用與註腳**：資料來源以隨頁註腳標記；各篇末詳列參考文獻。
- [ ] **圖表與圖片**：圖隨文；圖說加註拍攝者／提供者與拍攝時間。
- [ ] **數字**：計量與統計用阿拉伯數字；超過三位以半形逗號分節，如 1,234,567。
- [ ] **書名與篇名**：專書、報刊、期刊、影片、法條名用《》；單篇文章、論文篇名用〈〉。
- [ ] **人物**：人物篇採「生人不立傳」；志中提及人物直書姓名或官銜，不另加尊稱。
- [ ] **大事記**：編年紀事，按月、日排列；無詳細日期者以年為主，列於該年之前。

## 二、撰寫格式檢核

### 2.1 通用

- [ ] **概述與參考文獻**：各篇前有約 1,000–2,000 字概述；篇末詳列「參考文獻」。
- [ ] **標題階層**：依序為 篇 → 章 → 節 → 一 → （一）→ 1 → （1）→ A → a。
- [ ] **標點**：「」一般引文、『』引文內之引文、《》書報檔案名、〈〉論文及單篇文章篇名。
- [ ] **人名**：首次出現須加生卒年；西人加原文，如 連橫（1878-1936）、蘭克（Leopold von Ranke, 1795-1886）；生卒不詳者標（生卒不詳）。

### 2.2 正文

- [ ] **書寫**：橫式、正體中文；「臺」為繁體；外語專有名詞依原文，必要時加中文對照。
- [ ] **年代**：民國為主、西元輔之，如 民國 101 年（2012）；大範圍如「民國九○年代」。西元括號為全形（）。民國 34 年以前以政權年號為主，如 日治大正元年（1912）、清光緒 34 年（1908）。
- [ ] **括號與引號**：括號全形（）；雙括號（〔〕）；引號「」；雙引號「『』」。
- [ ] **數字**：年代、日期、歲數、統計用阿拉伯數字（如 102 年、105 年 1 月 1 日、18 歲、10,000 人）；表述性如「民國九○年代」「第二次世界大戰」用中文。
- [ ] **圖表**：表題在上、圖題在下；圖表均需編號，以「篇」為單位，如 表 2-1、圖 2-1；內文引用以編號書寫，勿用「如前圖」「見右表」。
- [ ] **獨立引文**：標楷體、每行前空三格、與正文上下各空一行。
- [ ] **引文**：正文引文加「」；內有引文用『』；原文有誤加（原誤）；節略以……標示。
- [ ] **註腳位置**：註腳號碼在標點符號前；引號「」內整段引文之註腳號碼置於引號內。

### 2.3 註腳

- [ ] 區分「說明註」與「參考註」；參考註應書寫完整引用資訊。
- [ ] 註腳字級 10；中文標楷體、外文 Times New Roman。
- [ ] 同章內同一文獻再次出現可簡化（如僅保留書名與頁碼）；首次出現須完整。
- [ ] 參考註之年份以西元標示。

### 2.4 參考註腳格式（抽檢）

- [ ] **專書**：作者，《書名》（出版地：出版者，出版年），起訖頁。
- [ ] **期刊**：作者，〈篇名〉，《刊名》，第 X 卷第 X 期（年月），起訖頁。
- [ ] **學位論文**：作者，〈篇名〉（學校地：系所，畢業年），起訖頁。
- [ ] **論文集**：作者，〈篇名〉，收入《書名》（出版地：出版者，出版年），起訖頁。
- [ ] **網路**：作者，〈篇名〉，收錄於「網站名稱」：網址（年/月/日點閱）。

---

## 三、檢核產出

1. **逐項對照**：依上述清單檢查指定內容，標示符合（✓）與不符合（✗）。
2. **列出問題**：對不符合項目註明位置（段落／圖表／註腳編號等）與違反條目。
3. **修正建議**：依凡例與撰寫格式寫出具體修改方式（含範例句式或格式範例）。
4. **摘要**：簡短總結整體符合程度與優先建議修正項目。
